Pflanzenkompendium
Einmalige Einrichtung. Alle Daten werden nur in deinem Browser gespeichert.
Apps Script URL
Im Apps Script Editor: Bereitstellen → Als Web-App bereitstellen → URL kopieren
Anthropic API Key
console.anthropic.com → API Keys → Create Key
Erste Nutzung: Pflanzendaten importieren
Importiert die 32 Pflanzen in dein Google Sheet. Nur einmalig nötig.
Wartung: Pflegepläne neu generieren
Holt für alle Pflanzen einen frischen Pflegeplan von der KI. Bestehende Intervall- und Event-Einstellungen werden NICHT überschrieben - nur leere Felder werden neu gefüllt. Dauer ca. 20-30 Sekunden.
🪴
Umtopfen
Frischer Substrat-Mix für die Pflanze. Gieß- und Düngedatum werden zurückgesetzt, Pflegeplan wird neu generiert beim nächsten Aufruf.
🧪
Dünger zuordnen
Die KI hat passende Pflanzen vorgeschlagen. Wähle aus, welche aktualisiert werden sollen. Nicht-gedüngte Pflanzen (Düngeintervall = 0) werden automatisch übersprungen.
